# Гайд по админке

## Быстрый старт
1. Откройте `/admin/connectors`.
2. Создайте коннектор (минимум: `Name`, `Chat ID`, `Price RUB`).
3. Скопируйте `Bot Link` из таблицы.
4. Откройте ссылку в Telegram и пройдите flow пользователя.

## Поля коннектора
- `Name`: название тарифа в боте.
- `Chat ID`: Telegram ID чата/канала в формате без минуса (`100...`), куда выдается доступ.
- `Price RUB`: сумма подписки.
- `Period days`: срок действия доступа в днях (по умолчанию `30`).
- `Start payload`: deeplink payload (`/start ...`). Если пусто, генерируется автоматически.
- `Internal ID`: внутренний ID коннектора. Если пусто, генерируется автоматически.
- `Offer URL`: ссылка на оферту.
- `Privacy URL`: ссылка на политику ПДн.
- `Description`: описание тарифа.

## Что обязательно
- Обязательно: `Name`, `Chat ID`, `Price RUB`.
- Опционально: остальные поля.

## Режим оплаты
- Пока платежный шлюз не выбран, используется `mock` режим.
- Кнопка `Оплатить` ведет на тестовую страницу `Mock Checkout`.

## Аудит действий
- Откройте `/admin/events`, чтобы увидеть последние действия пользователей.
- Там отображаются ключевые шаги: вход по `/start`, принятие условий, шаги анкеты, переход к оплате, mock-оплата.
