Connectors Guide

Гайд по админке

1. Как создать коннектор

Заполните Name, Chat ID и Price RUB. Нажмите Create connector.

2. Поля формы

Name: название тарифа, которое увидит пользователь.

Chat ID: Telegram ID чата/канала, например -1003626584986.

Price RUB: сумма оплаты в рублях.

Period days: длительность подписки в днях (по умолчанию 30).

Start payload: токен deeplink (/start ...). Если пусто, генерируется автоматически.

Internal ID: внутренний ID коннектора. Если пусто, генерируется автоматически.

Offer URL и Privacy URL: ссылки на оферту и политику ПДн.

Description: дополнительное описание тарифа.

3. Как тестировать

Скопируйте Bot Link из таблицы и откройте его в Telegram. Пройдите диалог и нажмите «Оплатить».

Пока активен mock-провайдер, откроется тестовая страница оплаты.
